how to get ibapi installed into project venv38:
1) download latest ibapi from ib
2) unzip into any folder
3) copy the pythonclient folder and save in the project directory
4) open Terminal
5) cd venv38
6) python -m pip install '../pythonclient'

example:
(venv38) (base) Tiger@linux-uc8n:~/PythonProjects/scottbrian_algo1/venv38> python -m pip install '../pythonclient'
